JavaScript examples

Form input and local storage

See the form component under 'Components'. Here a user can submit or delete comments. The local storage is used to save comments: See function addComment() from formDate.js . Try and enter comments and refresh the page!
Each comment is in its own div with its own delete button. To keep track of comments the local storage is used dynamicly and unique id's are used for each comment div
challenge is that the local storage can only save strings and not HTML elements which therefore is generated in the script together with the eventlisteners for the delete button as seen in the function updateCommSection() from formData.js.

Text input validation

In the form example under 'Components' the E-mail field is validatet. By using a keyup event the field can be validatet everytime some text is added to the field. By setting a variable 'true' or 'false' every time the field is validated we can check validity when pressing the submit button
