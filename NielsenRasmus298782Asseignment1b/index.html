Style Guide

Accessibility>>

Overall functionality and being able to understand content of a site no matter which kind of user as well as a well-structured page to help search-engines or screen-readers.

Colour Palette>>

Using eye-friendly and relevant colours as well as keeping contrast between elements combined with making a site readable by colourblind users.

Responsiveness>>

Having a dynamic website keeps the designer in charge of how the layout is represented when using different and odd sized browser windows.

Useability>>

Making a website easy to understand and navigate via e.g. symbols and intuitive functionality.
