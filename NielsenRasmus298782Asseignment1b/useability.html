Style Guide

Useability

Structure and spacing

Make the site easy to get an overview of. Using structure such as spacing between components and placement of content have a big impact on how the user sees the site. This site uses different spacing between the header and the navigation bar and between the navigation bar and the main content. Also, the main content is placed in the middle of the screen making the picture more balanced.

Easy to spot and understand links

A uniform design is used for links as well as making sure they are explained. All links on this page is underlined (and no other text is!) which brings easy understanding throughout the page.

Simplicity

Keep it simple is a help for the user as well as the designer. The design of this page is kept simple by using few colors and few different layouts for content. Also keeping the amount of info per page to a normal level helps, as seen in this project by splitting up topics into different pages and a single ‘Home’ page to give a brief summary of the topics.

Consistency

Consistency helps the user to understand different functionality. This page has a uniform header on all pages and boxes containing content all have the same layout.

Guidance

Make it easy to find your way on a page by explaining links and as in this project; keep a uniform header with navigation to all other sites on your domain. This can be more difficult with larger pages. In this project the page you are currently on will not be displayed in the navigation bar. Another approach could be to highlight it.

Typography

Another way to help a user differ between the type of content is by changing the typography as seen on this pages footer which is italic.

Relevancy

Thinking about who you target when designing a site. In this case the target would be web-designers at the same level as me, since the page is mostly a reference page for myself.

Reliability (not implemented)

In this context reliability is trustworthy content of a page. This could be to implement the ‘About Me’ page seen in the navigation bar with a description of my background, knowledge etc. It could also be to include a ‘job title’ or ‘job description’ field in a comment section.
