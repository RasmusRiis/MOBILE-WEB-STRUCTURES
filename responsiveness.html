Media queries

Media queries can set different design-rules to be used depending on screen width. Try for instance and make the site wider or thinner and watch how the navigation bar reacts, as well as the content boxes. This way the space is optimized when using smaller screens and content is kept together when using screen widths with plenty of space. This also works when zooming
